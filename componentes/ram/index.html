Memoria RAM

La memoria RAM es una memoria temporal y rápida que utiliza el ordenador mientras está encendido. Guarda datos e instrucciones que la CPU necesita usar en ese momento. Cuando se apaga el equipo, su contenido se pierde.

Funciones principales

  • Memoria temporal: almacena datos mientras los programas están funcionando.
  • Acceso rápido: permite que la CPU trabaje con información de forma ágil.
  • Multitarea: ayuda a tener varios programas abiertos al mismo tiempo.
  • Rendimiento: una cantidad insuficiente de RAM puede hacer que el equipo vaya lento.

Elementos relacionados

  • Módulo DIMM: formato habitual de RAM en ordenadores de sobremesa.
  • Ranura RAM: conector de la placa base donde se instala el módulo.
  • DDR: familia de tecnologías como DDR3, DDR4 o DDR5.
  • Frecuencia: velocidad de trabajo de la memoria.
  • Dual Channel: configuración que puede mejorar el rendimiento usando dos módulos compatibles.

Buenas prácticas

  • Comprobar que la RAM es compatible con la placa base.
  • Instalar los módulos en las ranuras recomendadas por el manual.
  • No forzar el módulo si no encaja correctamente.
  • Presionar hasta que las pestañas laterales queden cerradas.
  • Evitar tocar los contactos dorados del módulo.
