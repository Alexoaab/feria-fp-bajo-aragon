# Realidad Aumentada de Hardware · CPIFP Bajo Aragón

Aplicación web de Realidad Aumentada para visualizar componentes internos de un ordenador en 3D y participar en un reto de preguntas sobre hardware.

El proyecto está diseñado para el **stand de Informática del CPIFP Bajo Aragón**, donde el alumnado y los visitantes pueden explorar componentes informáticos mediante tarjetas o códigos de Realidad Aumentada.

---

## Objetivo de la aplicación

La aplicación permite:

- Visualizar modelos 3D de componentes informáticos.
- Usar tarjetas o códigos AR para mostrar los componentes.
- Manipular los modelos mediante controles de visualización.
- Aprender información básica de cada componente.
- Participar en un reto de preguntas.
- Registrar puntuaciones en un ranking.

---

## Componentes disponibles

La aplicación permite explorar los siguientes apartados:

- Equipo completo.
- Carcasa.
- Fuente de alimentación.
- Placa base.
- CPU / procesador.
- Refrigeración.
- Memoria RAM.
- Almacenamiento SSD.
- Tarjeta gráfica / GPU.
- Cables y conexiones internas.

---

## Uso en el stand

La aplicación está pensada para utilizarse en una **pantalla digital conectada por HDMI** a un ordenador.

Es necesario que la pantalla o el equipo utilizado disponga de **webcam**, ya que la aplicación necesita cámara para reconocer los códigos o tarjetas de Realidad Aumentada.

Funcionamiento recomendado en el stand:

1. Se conecta el ordenador a la pantalla digital mediante HDMI.
2. Se abre la aplicación en el navegador.
3. La aplicación se muestra en la pantalla del stand.
4. Los usuarios cogen las tarjetas o códigos AR.
5. Los usuarios mueven las tarjetas delante de la webcam.
6. Al mover la tarjeta, el componente 3D también cambia de orientación en pantalla.
7. El usuario, solo o con ayuda del staff del stand, puede ir pasando por los diferentes componentes desde el menú superior.

---

## Interacción con los modelos 3D

Cada componente puede visualizarse en Realidad Aumentada.

La aplicación incluye controles para:

- Aumentar el tamaño del modelo.
- Reducir el tamaño del modelo.
- Girar el modelo.
- Inclinarlo hacia arriba.
- Inclinarlo hacia abajo.
- Reiniciar el modelo a su posición inicial.

Estos controles ayudan a observar mejor los detalles de cada componente, como conectores, ranuras, ventiladores, cables o zonas importantes de montaje.

---

## Botón de información

Cada componente dispone de un botón de información.

Este apartado permite aprender:

- Qué es el componente.
- Para qué sirve.
- Dónde se instala.
- Qué relación tiene con el resto del ordenador.
- Qué aspectos conviene recordar durante el montaje o mantenimiento.

---

## Acceso desde móvil mediante QR

Además del uso en pantalla digital, los visitantes pueden acceder a la aplicación desde su propio móvil.

Para ello:

1. Escanean el código QR de acceso.
2. Abren la aplicación en el navegador del móvil.
3. Aceptan los permisos de cámara si quieren usar la Realidad Aumentada.
4. Pueden explorar componentes.
5. Pueden participar en el reto de preguntas.
6. Pueden consultar sus puntos y posición en el ranking.

---

## Reto de preguntas

La aplicación incluye un reto educativo de hardware.

Características del reto:

- El usuario introduce un alias.
- El cronómetro empieza cuando el usuario responde por primera vez una pregunta del reto.
- El reto completo tiene 50 preguntas.
- Las preguntas están repartidas entre los componentes.
- Gana quien consiga más aciertos.
- En caso de empate, gana quien haya tardado menos tiempo.
- La puntuación se guarda en el ranking.

El reto puede realizarse desde el móvil o desde la propia aplicación del stand.

---

## Ranking

La aplicación dispone de un sistema de ranking para consultar las puntuaciones de los participantes.

El ranking ordena los resultados teniendo en cuenta:

1. Mayor número de aciertos.
2. Menor tiempo en caso de empate.
